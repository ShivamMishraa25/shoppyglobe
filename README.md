## 📌 Project Overview

**ShoppyGlobe** is a modern e-commerce application where users can browse products, view detailed information, and manage their shopping cart. This project demonstrates component-based architecture, data fetching, routing, state management with Redux, and responsive UI styling.

---

## 🧰 Tech Stack

- **React JS** (Functional Components, Hooks)
- **Redux Toolkit** (Cart & Product State Management)
- **React Router DOM** (Page Navigation)
- **Axios** (API Data Fetching)
- **Tailwind CSS** (UI Styling & Responsiveness)
- **Framer Motion** (Smooth Transitions)

---

## 🔧 Features

- 🛒 **Product List:** Dynamically fetched from API (`https://dummyjson.com/products`)
- 🔍 **Search Functionality:** Real-time filtering of products
- 📦 **Add/Remove from Cart:** With quantity control via Redux
- 🧭 **Routing:** Home, Product Details, Cart, and Not Found page
- ⚡ **Performance Optimization:** Lazy loading components
- 📱 **Responsive Design:** Mobile and desktop-friendly UI
